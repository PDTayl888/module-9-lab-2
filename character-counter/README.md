When text is input the stats section will keep track of the character count and word count. 
In addition an average reading time will be calculated and displayed when text is present. 
A minimum and maximum word count can be set. When the word count is below the minimum and above the maximum
the word count text will appear in red, and white when within the min/max bounds.
In a similar manner a goal of reading time can be set as well and when the word count has reached the 
average reading time of the goal the text of the reading time goal will change from red to white. 
This is intended to be a tool for teachers to use in writing assignments.
